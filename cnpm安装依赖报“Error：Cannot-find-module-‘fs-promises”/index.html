cnpm安装依赖报Error：Cannot find module fs/promises

2021-08-22 1 min read # nodejs # 前端

出现这个错误，是由于cnpm版本跟node版本不匹配造成的，cnpm版本高了，node版本太低。

解决方案

1、升级nodejs版本，升级就不用说了吧，官网直接下载新版本的安装就行了。

2、降级cnpm版本，降级到7.1.0 npm install cnpm@7.1.0 -g 如果速度慢，加上 --registry=https://registry.npm.taobao.org
ok、搞定！

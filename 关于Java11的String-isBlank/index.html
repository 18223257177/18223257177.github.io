Java11的String.isBlank()

2021-05-22 1 min read # Java

昨天，我一个同事让我给帮忙看个问题，下载了一个项目下来，eclipse一直报红，说是找不到方法，我看是一个字符串直接使用了isBlank()，再看了一下eclipse版本，photon....额，确实有点老了，compiler都不支持11，isBlank是11新增的一个方法，判断字符串的。
简单说一下解决，要么就是换版本，升级IDE的版本（eclipse 18以后的就可以），以支持java11编译（前提是你安装的JDK11+）；要么就是使用StringUtils替代，但是这种很不科学，一旦代码里面使用地方多的话，改起来就相当麻烦。
所以还是建议升级吧，都21年了。全面小康了。。。。。。咱工具也得与时俱进~

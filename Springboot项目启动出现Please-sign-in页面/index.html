Springboot项目启动出现Please sign in页面

2020-02-11 1 min read # 后端 # Java

今天给同事写一个小工具，把以前的项目扒出来，写了几段代码。启动，打开浏览器，嗯？怎么出现了这页面。

一番搜索发现，原来这个是整合了security，然后又没有配置，所以才会出现这个页面。
解决方法就是：删除相关依赖。或者在启动类添加注解排除相关类：@SpringBootApplication(exclude = {SecurityAutoConfiguration.class,SecurityFilterAutoConfiguration.class})

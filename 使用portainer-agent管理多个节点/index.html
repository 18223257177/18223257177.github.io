使用portainer-agent配置管理多个节点

2021-03-22 1 min read # 服务器

简单几步，不罗嗦
登陆portainer后台，点击左侧endpoints菜单

然后就能看到一个节点的列表，点击Add endpoint按钮，进行添加

进入添加界面后，我们选择使用agent方式。

点击copy，复制那一段代码，到你需要被添加的服务器节点上去执行。
这里可以能会遇到一个问题，就是你直接执行这段的时候，可能不会成功，得先初始化swarm:

docker swarm init

创建agent成功之后，将服务器信息填入下面的输入框，name就是节点名字，自己取一个方便好记得，URL为你被添加服务器的IP:9001，后面的都可以默认，点击最下方的添加即可。
接着点击左侧home菜单，回到首页，即可看到添加的节点

如没有操作成功，欢迎提问，可以提供你需要的帮助~

mybatis的NoSuchMethodException异常是怎么回事

2020-08-17 1 min read # 后端 # Java

今天随机检查服务器日志，刚好发现一个异常，NoSuchMethodException: xxxx.(java.lang.String, java.lang.String, java.lang.String, java.util.Date···
这种看字面意思一眼就看出来错误所在了，也不知道是哪位小伙伴写的，粗心hhhh～
也罢，看到了，顺便就记录一下。这异常是说，XML在映射实体类的时候没有找到这个方法（也可能namespace没写对），这个实体类对应不上，构造参数不一致，先看看实体类有没有写带参构造，无参构造，然后一个一个数，参数顺序跟XML映射的也不能错。
这种只要细心一般就不会出现。

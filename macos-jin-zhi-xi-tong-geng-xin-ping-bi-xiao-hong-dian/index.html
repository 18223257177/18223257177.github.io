MacOS禁止系统更新，屏蔽小红点

2025-07-02 1 min read # Mac

1、忽略特定大版本更新，不同版本号，命令不同，仅供参考

sudo softwareupdate --ignore "macOS Big Sur"

2、取消红点

适用于 macOS 13 Ventura 及更高版本，依次在终端中执行以下三条命令：

defaults delete com.apple.systempreferences AttentionPrefBundleIDs

defaults delete com.apple.systempreferences DidShowPrefBundleIDs

killall Dock

适用于 macOS 11~12 的旧版命令：

defaults write com.apple.systempreferences AttentionPrefBundleIDs 0

killall Dock

3、禁止系统更新

sudo defaults write /Library/Preferences/com.apple.SoftwareUpdate AutomaticCheckEnabled -bool FALSE

sudo softwareupdate --schedule off

4、修改hosts，彻底屏蔽

sudo vim /etc/hosts

在文件中加入以下内容：

127.0.0.1 swscan.apple.com
127.0.0.1 swdist.apple.com
127.0.0.1 swcdn.apple.com

需要注意，写入了这些hosts，你的softwareupdate安装工具可能会有影响，请先安装好后再进行屏蔽更新操作。
完事！

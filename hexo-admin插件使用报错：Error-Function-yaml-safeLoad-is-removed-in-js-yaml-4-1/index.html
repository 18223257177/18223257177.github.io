hexo-admin新建文章报错：Error: Function yaml.safeLoad is removed in js-yaml 4

2021-08-27 1 min read # nodejs

今天在写博客时，发现新建文章一直Loading，目测是报错了，打开日志一看果然，提示：Error: Function yaml.safeLoad is removed in js-yaml 4. Use yaml.load instead, which is now safe by default.
大概就是这么个意思：safeLoad这个函数在js-yaml4这个版本已经被移除了，使用load替代。

看懂了然后就好办了，根据错误日志： at getSettings (F:\imem-blog\node_modules\_hexo-admin@2.3.0@hexo-admin\api.js:44:26)
定位到报错行44行，发现使用的果然是yaml.safeLoad,把它修改成yaml.load，ok重启hexo，打开后台再次新建文章，搞定！
